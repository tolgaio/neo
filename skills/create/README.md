## Usage

Use the `/create-{skill-name}` command to invoke a skill.

For example:
- `/create-summary` - Creates a summary with main points and takeaways
- `/create-academic-paper` - Generates a LaTeX academic paper
- `/create-visualization` - Creates visual diagrams using various tools
