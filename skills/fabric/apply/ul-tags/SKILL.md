# Apply UL Tags

## Description
Analyzes content and applies relevant tags from a predefined taxonomy including topics like AI, cybersecurity, politics, science, philosophy, and more. Outputs a JSON object with applicable tags.

## Triggers
- "apply ul tags to this..."
- "tag this content..."
- "categorize this content..."
- "what tags apply to this..."
