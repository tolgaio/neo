# Explain Skills

Imported from [Fabric patterns](https://github.com/danielmiessler/Fabric/tree/main/data/patterns).

## Available Skills

| Skill | Description |
|-------|-------------|
| [code](./code/) | Expert coder that explains code, security tool output, and configuration files |
| [docs](./docs/) | Expert at understanding and explaining tool documentation and instructions |
| [math](./math/) | Math teacher that explains equations and concepts in easy-to-understand terms |
| [project](./project/) | Expert at explaining what projects do and how to use them |
| [terms](./terms/) | Creates glossaries with definitions and analogies for advanced terms |

## Usage

Use the `/explain-{skill-name}` command to invoke a skill.
