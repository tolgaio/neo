## Usage

Use the `/extract-{skill-name}` command to invoke a skill.

For example:
- `/extract-wisdom` - Extract comprehensive wisdom from content
- `/extract-ideas` - Extract all important ideas
- `/extract-predictions` - Extract predictions with verification methods
