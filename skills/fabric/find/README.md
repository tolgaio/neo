# Find Skills

Imported from [Fabric patterns](https://github.com/danielmiessler/Fabric/tree/main/data/patterns).

## Available Skills

| Skill | Description |
|-------|-------------|
| [female-life-partner](./female-life-partner/) | Analyzes relationship criteria to reveal what someone is truly seeking in a woman life partner, cutting through indirectness and abstraction |
| [hidden-message](./hidden-message/) | Uncovers overt versus hidden political messages in propaganda, speeches, and conversations using cynical analysis of underlying intentions |
| [logical-fallacies](./logical-fallacies/) | Identifies all types of logical fallacies in arguments and text, including formal and informal fallacies |

## Usage

Use the `/find-{skill-name}` command to invoke a skill.
