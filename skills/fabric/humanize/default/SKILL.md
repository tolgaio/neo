# Humanize

## Description
Rewrites AI-generated or overly formal text to sound natural, conversational, and relatable, using everyday language that feels like it was written by a real person.

## Triggers
- "humanize this..."
- "make this sound more human..."
- "rewrite this to sound natural..."
- "make this more conversational..."
