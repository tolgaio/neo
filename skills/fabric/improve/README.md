# Improve Skills

Imported from [Fabric patterns](https://github.com/danielmiessler/Fabric/tree/main/data/patterns).

## Available Skills

| Skill | Description |
|-------|-------------|
| [academic-writing](./academic-writing/) | Refines text in academic and scientific language using common words for the best clarity, coherence, and ease of understanding |
| [prompt](./prompt/) | Expert LLM prompt writing service that improves AI prompts based on prompt engineering best practices from OpenAI |
| [report-finding](./report-finding/) | Improves security findings from penetration test reports with comprehensive sections and professional formatting |
| [writing](./writing/) | Refines text to enhance clarity, coherence, grammar, and style while maintaining original meaning |

## Usage

Use the `/improve-{skill-name}` command to invoke a skill.

Examples:
- `/improve-academic-writing` - Refine text into academic voice
- `/improve-prompt` - Enhance an AI prompt using best practices
- `/improve-report-finding` - Improve a security finding report
- `/improve-writing` - Fix grammar and improve clarity
