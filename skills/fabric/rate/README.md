# Rate Skills

Imported from [Fabric patterns](https://github.com/danielmiessler/Fabric/tree/main/data/patterns).

## Available Skills

| Skill | Description |
|-------|-------------|
| [ai-response](./ai-response/) | Rate AI responses by comparing them to ultra-qualified human performance |
| [ai-result](./ai-result/) | Assess AI/ML/LLM work results using multi-dimensional quality rating system |
| [content](./content/) | Classify and rate content based on idea density and theme matching |
| [value](./value/) | Calculate Value Per Minute (VPM) metric for content |

## Usage

Use the `/rate-{skill-name}` command to invoke a skill.

Examples:
- `/rate-ai-response` - Rate the quality of an AI response
- `/rate-ai-result` - Assess AI work results with detailed scoring
- `/rate-content` - Classify and rate content quality
- `/rate-value` - Calculate value per minute for content
