# Summarize Skills

Imported from [Fabric patterns](https://github.com/danielmiessler/Fabric/tree/main/data/patterns).

## Available Skills

| Skill | Description |
|-------|-------------|
| [default](./default/) | General-purpose content summarizer with one-sentence summary, main points, and takeaways |
| [board-meeting](./board-meeting/) | Converts raw board meeting transcripts into polished, formal meeting notes that meet corporate standards |
| [debate](./debate/) | Analyzes debates and discussions to determine primary disagreements and what evidence would change minds |
| [git-changes](./git-changes/) | Creates clean summaries of what changed in a GitHub project over the last 7 days |
| [git-diff](./git-diff/) | Creates concise summaries for Git diffs using conventional commit standards |
| [lecture](./lecture/) | Extracts the most relevant topics from a lecture transcript with structured summaries and timestamps |
| [legislation](./legislation/) | Reads and summarizes complex political proposals and legislation with balanced analysis |
| [meeting](./meeting/) | Analyzes meeting transcripts and extracts key information including action items and decisions |
| [micro](./micro/) | Creates ultra-concise summaries with one-sentence summary, main points, and takeaways |
| [newsletter](./newsletter/) | Extracts the most meaningful and interesting content from newsletters |
| [paper](./paper/) | Provides comprehensive academic paper reviews including technical approach and methodology |
| [prompt](./prompt/) | Takes AI chat prompts and outputs a concise summary of the prompt's purpose |
| [pull-requests](./pull-requests/) | Summarizes pull requests to a coding project in human-readable format |
| [rpg-session](./rpg-session/) | Summarizes in-person role-playing game sessions with key events and character development |

## Usage

- `/summarize` - General-purpose summarizer (default)
- `/summarize-{skill-name}` - Specialized summarizer (e.g., `/summarize-paper`)
