# Markdown Callout Creator

## Description
Creates markdown callouts based on provided text. Determines the most appropriate callout type (NOTE, TIP, IMPORTANT, WARNING, or CAUTION) and formats the content accordingly.

## Triggers
- "md callout this..."
- "create a markdown callout for..."
- "format this as a callout..."
