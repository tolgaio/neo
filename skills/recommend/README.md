# Recommend Skills

Imported from [Fabric patterns](https://github.com/danielmiessler/Fabric/tree/main/data/patterns).

## Available Skills

| Skill | Description |
|-------|-------------|
| [artists](./artists/) | EDM expert who recommends artists and creates festival schedules based on your musical preferences |
| [pipeline-upgrades](./pipeline-upgrades/) | Security specialist who optimizes vulnerability checking pipelines with more efficient sequences |
| [talkpanel-topics](./talkpanel-topics/) | Creates professional talk and panel proposals for conferences based on expertise and interests |
| [yoga-practice](./yoga-practice/) | Experienced yoga instructor providing personalized sequences, meditation guidance, and holistic lifestyle advice |

## Usage

Use the `/recommend-{skill-name}` command to invoke a skill.
