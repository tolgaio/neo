# Review Skills

Imported from [Fabric patterns](https://github.com/danielmiessler/Fabric/tree/main/data/patterns).

## Available Skills

| Skill | Description |
|-------|-------------|
| [code](./code/) | Perform comprehensive code reviews as a Principal Software Engineer, providing clear, constructive feedback on correctness, security, performance, readability, and best practices. |
| [design](./design/) | Expert solution architect review of architecture designs, evaluating clarity, security, performance, scalability, data management, and maintainability. |

## Usage

Use the `/review-code` or `/review-design` command to invoke a skill.
