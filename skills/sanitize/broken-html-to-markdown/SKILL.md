# Sanitize Broken HTML to Markdown

## Description
Converts messy HTML (with mixed HTML and markdown) into clean, properly formatted markdown with custom styling tags for use with Vitepress. This pattern handles quotes, videos, callouts, blockquotes, asides, definitions, and notes.

## Triggers
- "sanitize this HTML..."
- "clean up this HTML to markdown"
- "convert this broken HTML to markdown"
- "fix this messy HTML"
