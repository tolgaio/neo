# Write Skills

Imported from [Fabric patterns](https://github.com/danielmiessler/Fabric/tree/main/data/patterns).

## Available Skills

| Skill | Description |
|-------|-------------|
| [essay](./essay/) | Writes clear and illuminating essays in a specified author's style |
| [essay-pg](./essay-pg/) | Writes essays in Paul Graham's concise, clear, and simple style |
| [hackerone-report](./hackerone-report/) | Generates concise bug bounty reports for HackerOne |
| [latex](./latex/) | Generates syntactically correct LaTeX code for documents |
| [micro-essay](./micro-essay/) | Writes short essays (max 250 words) in Paul Graham's style |
| [nuclei-template-rule](./nuclei-template-rule/) | Creates YAML Nuclei templates for vulnerability detection |
| [pull-request](./pull-request/) | Drafts detailed pull request descriptions from git diff output |
| [semgrep-rule](./semgrep-rule/) | Creates Semgrep YAML rules for code pattern detection |

## Usage

Use the `/write-{skill-name}` command to invoke a skill.
