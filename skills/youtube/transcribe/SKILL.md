# YouTube Transcribe

## Description

Transcribe YouTube videos by extracting subtitles and auto-generated captions using yt-dlp via Docker. Supports both clean text output (ideal for AI processing) and timestamped output.

## Triggers

- "transcribe this youtube video..."
- "get transcript for youtube..."
- "extract captions from this video..."
- "what does this youtube video say..."
